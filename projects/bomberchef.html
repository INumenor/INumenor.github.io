About the Project

BomberChef is a fast-paced competitive multiplayer cooking game for PC where players race to complete culinary tasks while competing against each other.

I was responsible for the core game systems including the AI opponents, multiplayer lobby, and database backend that tracks scores and player progress.

PlatformPC
GenreCompetitive Cooking
ModeMultiplayer
EngineUnity / C#

Contributions
What I Built
🤖
Task Assignment AI

Designed AI system that dynamically assigns and manages cooking tasks, creating varied and challenging gameplay scenarios for players.

👾
Enemy AI

Built competitive AI opponents with different difficulty levels and behaviors to challenge players in single and multiplayer modes.

⏱️
Timer Systems

Implemented synchronized game timers across networked sessions ensuring fair and consistent competitive rounds.

🕹️
Character Movement

Built responsive character controller optimized for the fast-paced cooking environment with smooth collision and interaction.

🏠
Multiplayer Lobby

Designed and implemented the full lobby system — room creation, matchmaking, ready states, and session handoff to gameplay.

🗄️
Database Integration

Connected game to backend database for persistent player scores, stats, and progress tracking across sessions.
